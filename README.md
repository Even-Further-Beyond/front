### Anime Characters Client (React/Redux, TypeScript, Apollo)

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

### To run:

Clone repository

```
npm install
```

Configure .env file

```
REACT_APP_API_URL=
REACT_APP_S3_URL=
```

```
npm start
```

### To build for production:

```
npm run-script build
```
